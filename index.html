개발자 들여다보기: 체계적인 아키텍처

PickForMe는 유지보수와 확장이 쉬운 구조를 목표로 설계되었습니다. 역할과 책임에 따라 명확하게 분리된 폴더 구조는 코드의 가독성을 높이고 안정적인 앱의 기반이 됩니다.

  • 📁 src
    • 📁 assets - 이미지, 폰트 등 정적 리소스
    • 📁 components - 재사용 가능한 공통 UI 컴포넌트
    • 📁 context - 로그인 세션 등 전역 상태 관리
    • 📁 lib - 외부 서비스(Supabase, Kakao) 연동
    • 📁 navigation - 화면 전환 및 네비게이션 구조 정의
    • 📁 screens - 앱의 실제 페이지(화면) 단위 UI
    • 📁 theme - 컬러, 폰트 등 디자인 시스템
  • 📄 App.js - 앱의 최상위 진입점
  • 📄 package.json - 프로젝트 의존성 및 스크립트 관리
